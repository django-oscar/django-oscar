Compatibility
~~~~~~~~~~~~~

Oscar 3.2.2 is compatible with Django 3.2, 4.0, 4.1 and 4.2 and Python versions 3.8, 3.9, 3.10 and 3.11

Support for Django 3.1 has been dropped. Support for Python 3.7 has been dropped.
